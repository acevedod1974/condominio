# Condominio

Proyecto sencillo de gestión para un condominio.

**Resumen:**

- Código principal en `app.py`.
- Dependencias en `requirements.txt`.

**Requisitos**

- Python 3.8+.

**Instalación (entorno local)**

1. Crear y activar un entorno virtual:

```bash
python -m venv .venv
source .venv/bin/activate   # Linux / macOS
.venv\Scripts\Activate.ps1 # Windows PowerShell
```

2. Instalar dependencias:

```bash
pip install -r requirements.txt
```

**Ejecutar**

```bash
python app.py
```

**Desarrollo**

- Editar `app.py` para modificar la lógica de la aplicación.
- Añadir tests y documentación según se requiera.

**Contribuir**

- Crea ramas a partir de `main` y abre pull requests.
